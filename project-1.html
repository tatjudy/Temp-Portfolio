A custom built WordPress website using HTML, CSS, JavaScript, and PHP.

Project Overview

Susan Jomha at Distinct Interior Design needed a fully redesigned website. Her previous website was built on an old WordPress template that hasn't been updated since 2013 and was running in PHP 5. As her website needs to display her large portfolio containing image galleries, her website was extremely slow to load and unresponsive on mobile.

In order to start the process of creating a brand-new website for her, I first created a wireframe using Adobe XD to fully plan out the design of the website and gain her approval before I began development. I created the website with speed, responsiveness, and ease of use in mind.

To reduce the page speed, I minimized the use of plugins and reduced the use of JavaScript throughout the site.

As more and more users begin to use their mobile phones to browse the internet instead of their computers, the need for a fully responsive website is imperative in order to captivate clients viewing on-the-go. I made sure to develop the website with a mobile-first approach, which has been tested in multiple browsers and viewports to ensure accessibility on all devices and platforms.

Opportunities

I started building this website in my sparetime after I graduated in 2021 without any prior industry experience. Looking back after 2 years, I see multiple opportunties for improvement, such as using a CSS framework (BootStrap or Tailwind), organizing my code into reusable componenets, and adding CSS animations to capture the attention of the audience. I wanted to acknowledge that while I am more confident with my coding, there is always room for opportunity to grow no matter how long you've been in the industry.
